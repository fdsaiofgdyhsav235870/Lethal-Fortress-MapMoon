## Before Reporting

### 1. Check if the bug is already known
- Check the [existing issues](https://github.com/fdsaiofgdyhsav235870/Lethal-Fortress-MapMoon/issues) to see if your bug has already been reported
- Review the mod's changelog in thunderstore for known issues

Ensure you:
- installed the latest version of the mod
- installed and updated [DawnLib](https://github.com/daivuk/DawnLib)
- Don't have conflicting mods,**my test found out that the LLL,lethal level loader,may break these mod and other moon mod as well.**

## 报告bug前

### 1. 检查是否有重复问题
- 查看 [已有问题](https://github.com/fdsaiofgdyhsav235870/Lethal-Fortress-MapMoon/issues) 确认你的 Bug 是否已被报告
- 查看thunderstore模组的更新日志CHANGLOG看看是否已经修复BUG

确保你：
- 已安装最新版本的模组
- 已安装并更新 [DawnLib](https://github.com/daivuk/DawnLib)
- 没有冲突的模组，**我本人测试发现那些LLL(LethalLevelLoader)依赖的模组会让这两冲突出问题**
